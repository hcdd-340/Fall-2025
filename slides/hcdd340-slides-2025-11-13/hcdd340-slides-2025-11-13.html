Badge

Notifying user about new content

Supported for a PWA

Badge

  • `navigator.setAppBadge”
    • Displays a badge
  • `navigator.clearAppBadge”
    • Clear a badge

Badge best practices

  • Some systems don’t support it (e.g., Chrome on Linux)
    • Check if the current system supports it
    • Do not rely solely on badges to inform new content availability
  • Use sparingly

Source

Setting a badge

// Check for support first.
if (navigator.setAppBadge) {
  // Display the text as a badge
  navigator.setAppBadge("B");
}

Source

Clearing the badge

// Check for support first.
if (navigator.clearAppBadge) {
  // Remove the badge on the app icon.
  navigator.clearAppBadge();
}

Enabling dark mode

body {
  background-color: white;
  color: black;
}
input {
  background-color: white;
  color: black;
  border-color: black;
}
button {
  background-color: black;
  color: white;
}
@media
(prefers-color-scheme: dark)
{
  body {
    background-color: black;
    color: white;
  }
  input {
    background-color: black;
    color: white;
    border-color: white;
  }
  button {
    background-color: white;
    color: black;
  }
}

Browsers provide default color palettes

color-scheme

  • Default color for
    • canvas surface
    • scrollbars and other interaction UI
    • form controls
    • other browser-provided UI (e.g., “spellcheck” underlines)

Source

Example

html {
  color-scheme: light;
}
@media (prefers-color-scheme: dark) {
  html {
    color-scheme: dark;
