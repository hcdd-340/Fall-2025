## Badge

* `navigator.setAppBadge"
    - Displays a badge

* `navigator.clearAppBadge"
    - Clear a badge

---

### Badge best practices

* Some systems don't support it (e.g., Chrome on Linux)
    * [**Check**]{.t-salmon} if the current system supports it
    * Do not rely [**solely**]{.t-salmon} on badges to inform new content availability

* Use sparingly

:::{.t-ref}
[Source](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/How_to/Display_badge_on_app_icon#badge_best_practices)
:::

---

### Setting a badge

```js
// Check for support first.
if (navigator.setAppBadge) {
  // Display the text as a badge
  navigator.setAppBadge("B");
}
```

:::{.t-ref}
[Source](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/How_to/Display_badge_on_app_icon#displaying_and_updating_the_badge)
:::

---

### Clearing the badge

```js
// Check for support first.
if (navigator.clearAppBadge) {
  // Remove the badge on the app icon.
  navigator.clearAppBadge();
}
```

:::{.t-ref}
[Source](https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/How_to/Display_badge_on_app_icon#displaying_and_updating_the_badge)
:::
