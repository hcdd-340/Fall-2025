Start and end lines for an item

 /* end at column line 4 */

grid-template-areas

  • Name areas of the grid
  • Place items onto those named areas
  • Very useful

grid-template-areas

.container {
    display: grid;
    grid-template-columns:
        repeat(4,1fr);
    grid-template-areas:
        "h h h h"
        "s c c c"
        "s f f f";
}
header {
    grid-area: h;
}

.sidebar {
    grid-area: s;
}

.content {
    grid-area: c;
}

footer {
    grid-area: f;
}

grid-template-areas

  • Every cell must be filled
  • repeating name will span cells
  • . (period) is for empty cell

Responsive images

Common pattern

img {
    /* Contained within the parent */
    max-inline-size: 100%;

    /* aspect ratio is maintained */
    block-size: auto;

    /* crop the image if needed to
    preserve aspect ratio */
    object-fit: cover;

    /* How to crop (i.e., where is the
    most important part) */

    object-position: top center;
}

Media queries

  • Selectively apply CSS rules
    • For different device properties (“query”)
  • @media type and (feature)
    • type: all (default), print, screen

Media queries


body {
  color: black;
  background-color: grey;
}

@media print {
  body {
    background-color: transparent;
  }
}
